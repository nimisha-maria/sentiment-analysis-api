# Sentiment Analysis API

A **RESTful API** for sentiment analysis of text using **Python**, **Flask**, and **Hugging Face Transformers**.  
Classifies text as **positive**, **negative**, or **neutral**.

---

## **Features**
- Easy-to-use API endpoint `/sentiment`  
- Uses `distilbert-base-uncased-finetuned-sst-2-english` model from Hugging Face  
- Returns **JSON response** with label and confidence score  
- Local testing via VS Code or Python scripts  
- Ready for cloud deployment (Heroku, Render, Hugging Face Spaces)  

---

## **Tech Stack**
- Python 3.x  
- Flask  
- PyTorch  
- Transformers (Hugging Face)  
- VS Code (for development and testing)
